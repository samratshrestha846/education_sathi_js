# education_sathi

Monorepo for an online education consultancy discovery and referral platform.

## Stack

- `apps/edu_sathi_backend`: Express.js + TypeScript
- `apps/edu_sathi_frontend`: React + Vite + TypeScript
- `packages/domain`: domain entities, interfaces, and use cases
- `packages/shared`: shared utilities and contracts
- NoSQL-ready architecture with repository pattern
- Vitest for TDD

## Workspace Layout

```txt
apps/
  edu_sathi_backend/
  edu_sathi_frontend/
packages/
  domain/
  shared/
```

## Next Steps

1. Install dependencies with `npm install`
2. Start implementing repositories and use cases from `packages/domain`
3. Add a database package for MongoDB when persistence starts
